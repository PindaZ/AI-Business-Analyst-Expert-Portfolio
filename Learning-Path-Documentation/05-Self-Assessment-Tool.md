# 05-Self-Assessment-Tool: Track Your AI-BA Journey

Use this tool to evaluate your current proficiency across the six core AI-BA domains. Score yourself from 1 to 5 for each domain.

## 📊 Proficiency Scale
- **1 (Novice)**: Awareness only; no practical experience.
- **2 (Beginner)**: Understands core concepts; can assist under supervision.
- **3 (Practitioner)**: Can execute tasks independently; uses tools effectively.
- **4 (Expert)**: Leads complex initiatives; designs custom frameworks.
- **5 (Master)**: Strategically transforms organizations; develops industry-wide standards.
