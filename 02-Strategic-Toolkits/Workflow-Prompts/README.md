# Workflow Prompt Templates: The AI-Expert BA's Agentic Library

This folder contains a comprehensive collection of 28 specialized "Agentic Prompts" designed to automate and augment the daily workflows of an AI Business Analyst. These prompts utilize advanced techniques like **Chain-of-Thought**, **Few-Shot Learning**, and **Persona-based Constraints** to deliver enterprise-grade results.

## 🚀 How to Use These Prompts
1.  **Select your Phase**: Choose the prompt that matches your current project stage.
2.  **Fill in the Context**: Look for the `[Input Context]` tags and paste your specific project details.
3.  **Run in your LLM**: These are optimized for frontier models like **GPT-4o**, **Claude 3.5 Sonnet**, and **Llama 3**.
4.  **Refine & Iterate**: Use the "Prompt Debugger" if you need to tune the output for your specific domain.
